Match-Match Game

How to play :

  1. G

    When the game begins, all pictures are hidden

  2. A

    Select 2 cards to try to match the picture

  3. M

    If you match the pictures you can go again

  4. E

    If they don`t match the computer turn them

  5. !

    The player who finds all pairs wins!

This is a counter game where the object is to find pairs. You can also choose cards`s skirt and game difficult
